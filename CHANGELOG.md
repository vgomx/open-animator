# Changelog

## Unreleased

### Added
- SVG arc path import, basic CSS class styles, and animated mask support during playback
- Imperative playback transform driver for smoother large-scene preview
- Layer copy/paste with cross-tab clipboard persistence (`⌘C` / `⌘V`)
- Train parallax performance benchmark template
- Export size presets (Twitter, Instagram, favicon) in the export options dialog
- First-run keyboard shortcuts hint

### Changed
- Release stage promoted from alpha to beta
- Trackpad zoom sensitivity and playback preview smoothness restored
- README updated for current import/export capabilities

### Fixed
- Masked SVG layers now follow matrix animation during playback
- Large cached imports show a performance notice on load

## 2026-07-11

- Improved large SVG import performance and stabilized balloon regression tests
- UI polish: frosted chrome, rulers, templates, multi-artboard editing
